Notify IE users to upgrade their browser.

== Description ==

[See the plugin in action](http://pods.uproot.us) (only appears in IE6 and below)

This plugin notifies users that they should upgrade their browser. It adds a small pane to the top of the page alerting users to upgrade their browser. The pane can optionally be hidden by the user. The alert message can also optionally be changed.

== Installation ==

= Installation =
1. Unzip the upgrade-ie folder into wp-content/plugins/
2. Activate the plugin through Plugins > Installed
